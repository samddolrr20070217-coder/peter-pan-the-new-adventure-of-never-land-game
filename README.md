# peter-pan-the-new-adventure-of-never-land-game
HTML, CSS, JavaScript를 사용하여 피터 팬 애니메이션 모바일 시뮬레이터 액션 어드벤처 게임의 새로운 모험을 만들어 보세요. 플레이어는 왼쪽 패들을 위/아래 화살표 키로 조작하고, 컴퓨터는 오른쪽 패들을 공격 모드로 조작합니다. S를 누르면 비행모드로 전환 게임의 내용은 주인공 피터 팬이 웬디, 존, 마이클을 네버랜드로 데려가기 위해 런던으로 찾아오는 여정을 담고 있어야 합니다. 네버랜드에 도착하면 피터 팬은 잃어버린 아이들을 만나고, 뛰어놀고, 네버랜드 원주민들과 타이거 릴리를 만나고, 후크선장과 해적들과 싸우기도 합니다.
